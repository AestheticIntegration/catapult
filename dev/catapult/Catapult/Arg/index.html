Module Catapult.Arg

Custom arguments.

These arguments can be emitted as part of most events, and can be used to store custom data, debug messages, etc.

type t = [
  1. | `Int of int
  2. | `String of string
  3. | `Float of float
  4. | `Bool of bool
  5. | `Null
]
